1.DOM是什么

DOM是Document Object Model（文档对象模型 ）的缩写。

文档对象模型（Document Object Model，简称DOM），是W3C组织推荐的处理可扩展标志语言的标准编程接口。在网页上，组织页面（或文档）的对象被组织在一个树形结构中，用来表示文档中对象的标准模型就称为DOM。Document

2、js中常见的dom操作

A.查找dom节点

1.id查找 document.getElementById()

2.class查找 document.getElementsByClassName()

3.标签查找 document.getElementsByClassName()

使用childNodes;捕捉选定的dom节点的子集。

注意使用childNodes，捕捉子集后，会捕捉包括HTML节点，所有属性，文本节点。（回车，注释等等都会捕捉）

这个时候我们应该使用children;

B.操作节点（插入）

最常用的appendChild(), 添加节点至某dom节点的子集中最后一个

如果想把一个节点添加到指定的位置的话。

使用insertBefore(父节点，位置)

如果位置输入的父节点的某个子节点，会把节点添加到那个子节点的同胞节点，也就是在那个子节点的前面

appendChild和insertBefore的操作，只会插入节点，不会更改节点。
replaceChild 替换节点。
B.操作节点（移除）

removeChild()
